# Support

## How to Get Help

- **Usage questions:** Open a GitHub Discussion (if enabled) or an issue using the support template.
- **Bug reports:** Open a GitHub issue using the Bug Report template.
- **Feature requests:** Open a GitHub issue using the Feature Request template.
- **Sensitive concerns:** Open an issue with minimal details and maintainers will coordinate follow-up.

## Before Opening an Issue

Please check:

1. The [README](README.md) for setup and usage examples
2. Existing open and closed issues
3. The latest release notes in [CHANGELOG.md](CHANGELOG.md)

## Information to Include

When asking for help, include:

- OS and version
- Rust version (`rustc --version`)
- Command you ran
- Expected behavior and actual behavior
- Relevant logs or error output

## Response Expectations

This project is maintained on a best-effort basis. We aim to acknowledge new issues within a few business days.
